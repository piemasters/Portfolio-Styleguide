Source: styles/_variables.scss, line 1

5.1.1 Page Widths

The width of each different sized pages. These values are used to apply specific styling for set sized media devices. Styling by default targets mobile devices (sm). More specific styles applied are applied for md and lg+ devices.

Example

- $xs: 480px;
- $sm: 576px;
- $md: 768px;
- $lg: 992px;
- $xl: 1200px;
- $xxl: 1800px;
